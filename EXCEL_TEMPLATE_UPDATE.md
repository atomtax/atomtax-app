# 📊 매매사업자 물건목록 업로드 양식 변경 완료

## ✅ 변경 사항 요약

### 1️⃣ **물건목록 시트 컬럼 변경**

**기존 컬럼 (13개)**:
```
사업자번호* | 물건명 | 소재지 | 취득가액 | 기타필요경비 | 양도가액 | 처분비 | 양도소득 | 양도일(YYYYMMDD) | 신고기한 | 85초과(Y/N) | 진행단계 | 비고
```

**신규 컬럼 (9개)** ✅:
```
사업자번호* | 물건명 | 소재지 | 취득일(YYYYMMDD) | 양도일(YYYYMMDD) | 양도가액 | 신고기한 | 85초과(O/X) | 비고
```

**제거된 컬럼**:
- ❌ 취득가액 (필요경비 상세에서 자동 계산)
- ❌ 기타필요경비 (필요경비 상세에서 자동 계산)
- ❌ 처분비 (사용 안 함)
- ❌ 양도소득 (자동 계산)
- ❌ 진행단계 (시스템에서 관리)

**추가된 컬럼**:
- ✅ 취득일(YYYYMMDD)

**변경된 컬럼**:
- 85초과(Y/N) → **85초과(O/X)**

---

### 2️⃣ **필요경비 상세 시트 - 번호/비용명 고정**

**변경 전**: 사용자가 직접 입력
**변경 후**: 드롭다운으로 선택

**비용명 선택지**:
1. 취득가액
2. 취득세 등
3. 중개수수료
4. 신탁말소비용
5. 관리비 정산
6. 기타비용

**번호**: 1, 2, 3, 4 순서대로 자동 표시

---

### 3️⃣ **85초과 표시 방식 변경**

**변경 전**: Y/N
**변경 후**: **O/X**

- O → 85초과 (Y로 저장)
- X → 85미만 (N으로 저장)
- Y/N 입력도 호환 가능

## 🔧 업로드 로직 변경

### **자동 계산 항목**:
- ✅ **취득가액**: 필요경비 상세에서 "취득가액" 구분 합계
- ✅ **기타필요경비**: 필요경비 상세에서 "기타필요경비" 구분 합계
- ✅ **양도소득**: 양도가액 - 취득가액 - 기타필요경비
- ✅ **신고기한**: 양도일이 속한 달의 말일로부터 2개월 후

### **필수 입력 항목**:
1. **사업자번호*** (필수)
2. **물건명** (필수)
3. **양도일(YYYYMMDD)** (필수)
4. **양도가액** (필수)

### **선택 입력 항목**:
- 소재지
- 취득일(YYYYMMDD)
- 신고기한 (없으면 자동 계산)
- 85초과(O/X) (기본값: X)
- 비고

## ⚠️ 주의사항

### **1. 포함되지 않는 컬럼은 무시**
- 엑셀 파일에 다른 컬럼이 있어도 시스템에서 읽지 않습니다
- 오직 위 9개 컬럼만 인식합니다

### **2. 날짜 형식**
- 반드시 **YYYYMMDD** 형식으로 입력 (예: 20250115)
- 하이픈(-)이나 슬래시(/) 있어도 자동 제거

### **3. 85초과 입력**
- O 또는 X 입력
- Y 또는 N 입력도 호환됨
- 빈 값은 X로 처리

### **4. 필요경비 상세**
- 물건명은 물건목록의 물건명과 **정확히 일치**해야 함
- 번호는 순서대로 1, 2, 3, 4...
- 비용명은 정해진 항목 사용 권장
