## 💡 자동 계산 로직

### 양도소득 계산식:
```
양도소득 = 양도가액 - 취득가액 - 기타필요경비
```

**예시:**
```
양도가액: 150,000,000원
취득가액: 100,000,000원
기타필요경비: 5,000,000원
─────────────────────
양도소득: 45,000,000원 (자동 계산)
```
