## 🚀 사용 방법

### 기본 실행

```bash
# 1. 폴더로 이동
cd C:\내부페이지\홈택스자동화

# 2. 프로그램 실행
python hometax_simple.py
```

### 실행 과정

1. **다운로드 경로 입력** (Enter = 기본값)
2. **연도 입력** (Enter = 올해)
3. **월 입력** (Enter = 이번 달)
4. **Chrome 자동 실행** → 홈택스 접속
5. **공인인증서 로그인**
6. **CMD에서 Enter**
7. **신고자료 다운로드**
8. **CMD에서 Enter**
9. **파일 자동 정리 완료!**

## 📅 자동 실행 설정 (선택사항)

매일 자동으로 실행하고 싶다면 Windows 작업 스케줄러를 사용하세요.

### 간단 설정 방법

1. **시작 → "작업 스케줄러" 검색**
2. **작업 만들기 클릭**
3. **일반 탭:**
   - 이름: `홈택스 자동 다운로드`
   - 설명: `매일 오전 9시 신고자료 다운로드`
4. **트리거 탭:**
   - 새로 만들기
   - 매일 오전 9:00
5. **동작 탭:**
   - 새로 만들기
   - 프로그램: `python`
   - 인수: `hometax_simple.py`
   - 시작 위치: `C:\내부페이지\홈택스자동화`
6. **확인 클릭**

## 🔧 문제 해결

### 문제 1: "pip 명령어를 찾을 수 없습니다"

**해결:**
```bash
python -m pip install webdriver-manager
```

### 문제 2: Chrome이 열리지 않음

**해결:**
- Chrome 브라우저가 설치되어 있는지 확인
- Chrome 최신 버전으로 업데이트

### 문제 3: 공인인증서 로그인 안 됨

**해결:**
- 수동으로 로그인 버튼 클릭
- 공인인증서가 올바르게 설치되어 있는지 확인

### 문제 4: 파일이 다운로드 안 됨

**해결:**
- 홈택스에서 수동으로 다운로드 버튼 클릭
- 다운로드 폴더 경로 확인

---

## 📊 예상 효과

### 시간 절약
- **수동 작업:** 1회 10분 × 월 20회 = 200분/월
- **자동화 후:** 1회 2분 × 월 20회 = 40분/월
- **절약:** 160분/월 (약 2.7시간)

### 편의성
- ✅ 자동 로그인
- ✅ 자동 다운로드 경로 설정
- ✅ 자동 파일 정리 (연도/월별)
- ✅ 중복 다운로드 방지
