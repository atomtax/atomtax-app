## 📋 3가지 조회 방법

### 1️⃣ **거래처명 클릭** → 빠른 조회 팝업
- 클릭 즉시 팝업으로 정보 확인
- 페이지 이동 없음
- 빠른 조회에 최적화

### 2️⃣ **👁️ 아이콘 클릭** → 빠른 조회 팝업
- 거래처명 클릭과 동일
- 시각적으로 명확한 표시

### 3️⃣ **🗗 아이콘 클릭** → 새 탭에서 상세 페이지
- 새 창에서 열림
- 기존 목록 유지
- 상세 정보 확인/수정

---

## 🎨 빠른 조회 팝업 특징

### 📑 탭 구조
```
┌─────────────────────────────────────────┐
│  금아실업                          [✕]  │
├─────────────────────────────────────────┤
│  [기본정보] [사업자정보] [세무정보]     │
├─────────────────────────────────────────┤
│                                         │
│  ✓ 선택된 탭의 정보만 표시              │
│  ✓ 깔끔한 정보 구성                     │
│  ✓ 빠른 로딩                            │
│                                         │
├─────────────────────────────────────────┤
│        [닫기] [상세 페이지로 이동]      │
└─────────────────────────────────────────┘
```

### 1. **기본정보 탭**
- 번호
- 거래처명
- 담당자
- 대표자
- 연락처 (포맷팅)
- 이메일

### 2. **사업자정보 탭**
- 사업자번호 (포맷팅)
- 주민등록번호 (포맷팅)
- 법인등록번호 (포맷팅)
- 업태
- 종목
- 업종코드
- 우편번호
- 사업장 주소

### 3. **세무정보 탭**
- 공급가액
- 세액
- 최초출금월
- 홈택스 아이디
- 홈택스 비밀번호

## 🎯 사용 시나리오

### 시나리오 1: 빠른 확인
```
상황: 전화 통화 중 고객사 정보 확인 필요
방법: 거래처명 클릭 → 팝업에서 즉시 확인
장점: 페이지 이동 없이 빠른 조회
```

### 시나리오 2: 상세 작업
```
상황: 고객사 정보 수정 필요
방법: 🗗 아이콘 클릭 → 새 탭에서 상세 페이지
장점: 목록 유지하면서 여러 고객사 동시 작업 가능
```

### 시나리오 3: 비교 작업
```
상황: 여러 고객사 정보 비교 필요
방법: 🗗 아이콘을 여러 번 클릭 → 여러 탭에서 동시 확인
장점: 탭 전환으로 쉽게 비교
```

## 💡 UI/UX 개선사항

### 1. **시각적 구분**
- 👁️ 아이콘: 파란색 (#667eea)
- 🗗 아이콘: 회색 (#6b7280)
- 호버 시: 배경색 변경 + 크기 증가

### 2. **빠른 접근**
- 거래처명 자체도 클릭 가능
- 아이콘으로 명확한 기능 표시
- 툴팁으로 기능 설명

### 3. **탭 전환 애니메이션**
- 부드러운 페이드인 효과
- 탭 버튼 활성화 표시
- 직관적인 네비게이션

### 4. **모달 배경 클릭**
- 배경 클릭 시 팝업 자동 닫힘
- ESC 키로도 닫기 가능 (추가 가능)
