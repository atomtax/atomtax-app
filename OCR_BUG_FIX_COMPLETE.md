## 🐛 기존에 발견된 문제 (구버전)

### 1️⃣ 취득가액 2번 입력 문제
**원인**: 매각대금 완납증명원에서 `acquisition_value`와 `expenses`에 모두 취득가액을 추가하여 중복 입력됨

**상태**: ✅ **완전 해결됨** (V2)
- 필요경비 상세에도 추가되지만
- 중복 파일 업로드 시 최대값만 선택
- 필요경비 중복 제거 로직 추가

---

### 2️⃣ 소재지 입력 안 됨 문제
**원인**: `address` 필드에는 입력되지만, 물건 행이 자동으로 확장되지 않아 사용자가 보지 못함

**상태**: ✅ **해결됨**
- `toggleDetailRow(propertyIndex)` 호출로 자동 확장
- 상세 로그 추가로 입력 확인 가능

---

### 3️⃣ 필요경비 상세 입력 안 됨 문제
**원인**: `expenses` 배열에는 추가되지만, 필요경비 섹션이 자동으로 열리지 않아 사용자가 보지 못함

**상태**: ✅ **완전 해결됨** (V2)
- 매각대금 완납증명원도 필요경비 상세에 추가
- 0.5초 후 필요경비 섹션 자동 열기
- 8종 서류 모두 필요경비 상세 지원
